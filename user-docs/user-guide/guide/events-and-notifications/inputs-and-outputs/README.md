# Inputs and outputs

The **Inputs and outputs** section covers alert rules for monitoring connected sensors and equipment on your GPS devices.

- [**Input triggering**](input-triggering.md): Alerts when a discrete device input changes state, such as a door sensor or ignition.
- [**Output triggering**](output-triggering.md): Monitors output state changes such as engine block or alarm activation.
- [**Parameter in range**](parameter-in-range.md): Alerts when a sensor reading enters or exits a defined value range.
- [**State field value**](state-field-value.md): Notifies when a device state field reaches an expected value.
