# Device positioning

The **Device positioning** section covers events related to GPS signal and satellite data for your devices.

- [**GPS antenna disconnected**](gps-antenna-disconnected.md): Alerts when an external GPS antenna is disconnected from a device.
- [**GPS jamming**](gps-jamming.md): Detects GPS or GNSS signal interference used to disable tracking.
- [**GPS signal loss and recovery**](gps-signal-loss-and-recovery.md): Notifies when a device loses its GPS signal and when it recovers.
- [**Location report on demand**](location-report-on-demand.md): Requests a device's current location via SMS when standard IP connections are unavailable.
