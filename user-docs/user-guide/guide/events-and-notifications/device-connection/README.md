# Device connection

The **Device connection** section covers events related to GPS device connectivity to the cellular network and the Navixy platform.

- [**Device lost connection**](device-lost-connection.md): Notifies you when a GPS device goes offline beyond your configured timeout.
- [**GSM jamming**](gsm-jamming.md): Detects when GSM cellular signals are jammed, preventing data transmission to the platform.
