# Vehicle efficiency

The **Vehicle efficiency** section covers alert rules that help monitor engine
status, idling, and fuel use across a fleet.

- [**Check engine (MIL)**](check-engine-mil.md): Alerts when the Malfunction
  Indicator Lamp (check engine light) activates on a vehicle.
- [**Excessive idling**](excessive-idling.md): Alerts when a vehicle idles
  longer than a set threshold, using platform logic or hardware events.
- [**Fuel level change**](fuel-level-change.md): Alerts when the fuel sensor
  detects a significant level change, signaling refueling or potential theft.
